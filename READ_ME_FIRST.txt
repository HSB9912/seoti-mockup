🎨 STEP 2 · 결정해야 할 항목
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① 메인 스토어 디자인  →  B로 확정 ✅
                         (모던 미니멀)

다음 항목들만 골라주시면 됩니다:

② 대량주문 페이지     →  A / B / C 중 하나
③ Q&A(문의) 페이지    →  A / B 중 하나
④ 관리자 페이지       →  A / B / C 중 하나
⑤ 상품 상세 페이지    →  A / B 중 하나
⑥ 장바구니·결제       →  A / B 중 하나  (비회원 결제 가능)
⑦ 로그인·회원가입     →  A / B 중 하나
⑧ 마이페이지          →  A / B 중 하나
⑨ About(작가 소개)    →  A / B 중 하나
⑩ 공지사항·이벤트     →  A / B 중 하나

각 변형은 layout(화면 구조), 사용 방식,
시각 스타일이 모두 다릅니다.

🛒 ② 대량주문 페이지
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

손님이 "이거 20개, 저거 30개 살게요" 하고
견적 신청하는 화면입니다.

[A] 다단계 위저드형
    1단계→2단계→3단계 순서대로 진행
    상품 그리드 + 따라다니는 카트
    단계가 명확해서 헷갈리지 않음
    화이트·미니멀 톤

[B] 단일 페이지 폼
    한 화면에서 정보입력 + 상품선택
    상품이 가로 리스트형(테이블 같은)
    한 번에 다 보임, 정보 빽빽한 편

[C] 드로어(슬라이드) 방식
    상품 카드의 [+ 담기] 누르면
    오른쪽에서 장바구니가 슥 슬라이드
    따뜻한 파스텔, 모바일 친화적


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💬 ③ Q&A 페이지
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[A] 카드 리스트
    Q/A를 큰 카드로 시원하게
    클릭하면 답변이 아래로 펼쳐짐
    친근·캐주얼한 느낌

[B] 게시판형
    흔한 고객센터 스타일
    왼쪽에 카테고리·상태 필터
    표 형태로 정보 빽빽


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚙️ ④ 관리자 페이지
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

관리자만 보는 페이지. 재고/매출/상품등록/
대량주문 처리/Q&A 답변 + 회원·리뷰·공지·쿠폰
관리까지 다 여기서.

[A] 클래식 사이드바형
    왼쪽 메뉴 + 표 위주
    흔한 어드민 스타일, 안정적

[B] 상단 네비 + 카드형
    위쪽 가로 메뉴
    둥근 카드 + 선 그래프 + 칸반 보드
    Linear/Notion 같은 모던한 느낌

[C] 노션형 트리
    슬림한 좌측 트리 메뉴
    여백 많고 콘텐츠 중심
    극단적 미니멀


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📦 ⑤~⑩ 추가 페이지 (각 A / B 변형)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⑤ 상품 상세 페이지 (개별 상품 화면)
   - 리뷰 시스템 포함 (별점·사진리뷰·작가답글)
   - 재입고 알림 기능
   A: 좌측 갤러리 + 우측 정보판 (정적)
   B: 가로 슬라이드 갤러리 + 정보 하단 (역동적)

⑥ 장바구니 → 결제 흐름
   - 비회원 결제 가능
   - 회원 결제도 가능
   A: 다단계 (장바구니→주문서→결제→완료)
   B: 단일 페이지 (한 화면에서 모든 입력)

⑦ 로그인 / 회원가입
   A: 탭 전환식 (로그인↔회원가입↔비번찾기)
   B: 좌우 분할(브랜드 비주얼 + 폼)

⑧ 마이페이지
   주문조회 · 위시리스트 · 적립금 ·
   쿠폰 · 리뷰 · 회원정보
   A: 좌측 사이드바 메뉴
   B: 상단 탭 + 대시보드 카드

⑨ About (작가 소개)
   A: 긴 글 스크롤 (에디토리얼)
   B: 갤러리/포트폴리오형 (이미지 중심)

⑩ 공지사항 / 이벤트
   A: 리스트형 + 상단 강조 배너
   B: 카드 그리드형

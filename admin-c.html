새 상품 추가

대표 이미지를 끌어다놓고 정보 입력 후 ＋ 등록. R2 에 즉시 업로드되고 Supabase products 테이블에 저장돼요.

Group Order 견적 아직 견적 요청 없음

고객이 보낸 견적 요청을 한 줄씩 확인하고, 클릭해서 금액·배송정보를 입력 후 회신할 수 있어요.

📋
아직 도착한 견적 요청이 없어요

고객이 손님 사이트 Group Order에서 견적을 보내면 여기에 표시됩니다.

💬

Q&A 답변 아직 문의 없음

평일 기준 1~2일 내 답변을 목표로 운영. 자주쓰는 답변 등록해두면 클릭으로 삽입 가능합니다.

💬
아직 문의가 없어요

고객이 QnA에 글을 남기면 여기에서 답변할 수 있습니다.

📈

판매 실적 아직 데이터 없음

주문이 들어오면 매출·국가·카테고리·재방문율 등이 자동 집계됩니다. orders 테이블 연동 후 활성화됩니다.
